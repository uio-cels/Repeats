What the script does¶

WHAT IT DOES
Parse RepeatMasker outputs to get summary info for each repeat as well as masked amounts by class and family.
This is the first basic script to run - most labs have their own!

For the repeats, it will provide:
 - fragment number (frg nb): 
      all
      frg nb from start to end of consensus
      frg nb corrected for interupted repeats (using the "ID". e.g. from the Repeat Masker .out)
 - %div, ins, del: pondered average, median
 - length masked + %genome by this repeat
 - amount of DNA that is masked several times (usually 2) by this element 

Summary will provide amounts and % masked by various class, families
Plus the script provides a set of files with overlap info to help giving real amounts/%

def histogram(dataframe, fullframe):
    params = {'mathtext.default': 'sf' }          
    plt.rcParams.update(params)

    sns.set(context='paper', style = 'ticks', font = 'sans serif', font_scale=2)
    sns.plt.figure(figsize=(10,7))
    
    ax = plt.subplot(211)
    
    sns.distplot(dataframe)
    plt.title('A')
    plt.ylabel('Number of families')
    plt.xlabel('Fraction of genome (%)')
    plt.grid(False)
    plt.xlim(xmin = 0)
    plt.grid(False)
    
    ax = plt.subplot(212)
    
    sns.distplot(fullframe)
    plt.ylabel('Count')
    plt.xlabel('Average length masked (bp)')
    plt.grid(False)
    plt.xlim(xmin = 0)
    plt.grid(False)
    
    plt.tight_layout()

def overview(Class, coverage_minimum, parameter):
    fullframe = classified[classified['Rclass']==Class][[parameter]]
    CLASS = classified[classified['Rclass']==Class].groupby('Rfam').sum()[['%_GENOME']]
    histogram(CLASS, fullframe)
    print "Families covering more than %s percent of the genome:" % coverage_minimum   
    print CLASS[CLASS['%_GENOME']>coverage_minimum][['%_GENOME']]
